# Mohiddin Moghal - Professional Portfolio Website

This folder contains the regenerated professional portfolio website using your uploaded profile photo.

## Replace or update the profile photo

1. Add your updated photo to the `assets` folder.
2. Rename it exactly to `profile-photo.jpg`.
3. Replace the existing file at:

   `assets/profile-photo.jpg`

The website will automatically use the new image as long as the filename remains the same.

## Host on GitHub Pages

Upload the extracted files, not the ZIP, to your GitHub Pages repository. Keep this structure:

```text
index.html
styles.css
script.js
assets/
  profile-photo.jpg
  Mohiddin_Moghal_Professional_Masterpiece.pdf
  Mohiddin_Moghal_Professional_Masterpiece.pptx
```

## Included assets

- `index.html` - portfolio website
- `styles.css` - professional responsive styling and effects
- `script.js` - navigation, scroll progress, and reveal effects
- `assets/profile-photo.jpg` - your current profile photo
- `assets/Mohiddin_Moghal_Professional_Masterpiece.pdf` - updated portfolio PDF
- `assets/Mohiddin_Moghal_Professional_Masterpiece.pptx` - editable portfolio PowerPoint
